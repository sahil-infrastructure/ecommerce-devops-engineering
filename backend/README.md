# E-Commerce Platform Engineering Project

## Features

- Flask Backend
- PostgreSQL Database
- Docker & Docker Compose
- Health Checks
- cAdvisor Monitoring
- Prometheus Metrics Collection
- Grafana Dashboards
- Alerting Rules

- Deployment Rollback Simulation

Flask App -> PostgreSQL -> Docker Compose -> cAdvisor -> Prometheus -> Grafana
